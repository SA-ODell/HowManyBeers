How Many Beers?

It's pretty simple. You need X calories every day. Food accounts for Y calories. If you keep Y less than X, then you can apply what's left over for beer. Easy, yeah?

Baseline Metabolic Rate

There are several forumulas one can use to calculate daily caloric requirements, but they all focus on the same data points. Daily calories are a function of ones age, gender, BMI, and exercise expenditures. For this project, we're going to use a variation of the Mifflin - St Jeor equation. Google it.
